## Requisitos

* [Node v13.9.0 o reciente](https://nodejs.org/es/)
* [NPM v6.13.7 o reciente](https://www.npmjs.com/get-npm)

## Instalación

```bash
# Instalar las dependencias/librerías
$ npm install

# Inicializar ionic/angular
$ npm run start
```


## Tecnologías

El proyecto usa las siguientes librerias:

1. Ionic/Angular
2. Firebase
3. Sass
